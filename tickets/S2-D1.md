## Edge Cases

### 1. ISR can 2 requests

Khi test time-based ISR, request thu nhat co the chi trigger revalidation. Can request lai sau interval de thay du lieu moi. Day la behavior binh thuong, khong phai bug.

### 2. Windows-friendly import scan

Neu can verify runtime imports bang PowerShell, co the dung:

```powershell
Get-ChildItem src\app,src\components -Recurse -Include *.ts,*.tsx |
  Select-String -Pattern '@/lib/faqs|@/lib/pricing|@/lib/schedule|@/lib/locations|@/lib/site'
```

Expected: khong con runtime imports ngoai nhung exceptions da duoc document ro.

### 3. Type-only imports

`import type` tu static files co the van ton tai tam thoi neu da duoc document, nhung muc tieu uu tien la loai bo ca runtime dependency.

### 4. Fail criteria

Neu bat ky acceptance criterion nao fail, quay lai fix ticket nguon. Khong gop fix ngau nhien vao D1.
