# S2-C1: Wire 3 Existing Pages <- Sanity

## Objective

Chuyen 3 pages hien co sang doc Sanity thay vi static libs, theo cung pattern server fetch nhu homepage.

## Input

- `src/app/(site)/hoc-cau-long-cho-nguoi-moi/page.tsx`
- `src/app/(site)/lop-cau-long-binh-thanh/page.tsx`
- `src/app/(site)/lop-cau-long-thu-duc/page.tsx`
- Query helpers tu S2-A2

## Output

### `hoc-cau-long-cho-nguoi-moi`

- `getFaqs("nguoi_moi")`
- `getPricingTiers()`
- filter bo `enterprise` neu UI hien tai khong can

### `lop-cau-long-binh-thanh`

- `getFaqs("binh_thanh")`
- `getLocations()` roi filter `district === "binh_thanh"`

### `lop-cau-long-thu-duc`

- `getFaqs("thu_duc")`
- `getLocations()` roi filter `district === "thu_duc"`

## Dependency

- S2-A2

## Priority

- P1

## Acceptance Criteria

1. 3 pages render dung content tu Sanity.
2. `nguoi_moi` hien pricing cards va FAQs.
3. `binh_thanh` chi hien courts Binh Thanh va FAQs dung.
4. `thu_duc` chi hien courts Thu Duc va FAQs dung.
5. 3 pages khong con import static runtime data tu `faqs.ts`, `pricing.ts`, `locations.ts`.
6. `npm run build` pass.

## Edge Cases

### 1. FAQ answer rendering

3 pages hien dang render FAQ plain text. Khi chuyen sang Sanity, nen tai su dung Portable Text rendering pattern tu S2-B3 thay vi tu viet lai lan nua.

### 2. Filter local pages bang district

Sanity location schema khong co `localPage`. Canonical filter cho Sprint 2 la:

- `district === "binh_thanh"`
- `district === "thu_duc"`

### 3. JSON-LD tam thoi chua doi

Trong ticket nay, visible content se doc Sanity truoc. Structured data van co the doc static den khi S2-C2 hoan thanh. Day la compromise co chu dich va phai duoc xoa trong C2.

### 4. Metadata generation

`buildMetadata()` va route metadata chua nam trong scope ticket nay.

## Non-Scope

- Chua refactor JSON-LD builders
- Chua doi metadata generation
- Chua doi UI/layout
- Chua them pages moi

## Risk / Luu y

- Neu FAQ rendering bi duplicate logic giua homepage va 3 pages, nen extract helper nho hoac shared component ngay trong sprint nay.
